## Prerequisites

- rust (wasm32-wasm32-unknown target)
- go 1.20 or higher
- llvm-cov

## How to build

```bash
make install-dev

make build
```

## How to test

```bash
cargo test --workspace --exclude hpl-tests

cargo llvm-cov --workspace --exclude hpl-tests
```
